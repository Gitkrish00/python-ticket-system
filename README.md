# python-ticket-system
A Python-based cinema ticket and food ordering system with cart and billing functionality.

Desp: # Cinema Ticket & Food Ordering System (Python)

A simple command-line based cinema booking system built using Python.
Users can view available movies, select tickets, add food items, and get the total bill.

##  Features

* Display available movie tickets 
* Display food menu 
* Add items to cart 
* Calculate total bill 
* Simple and interactive console interface

##  Tech Used

## How to Use

* View available tickets and food items
* Type the exact item name to add to cart
* Add multiple items
* Type **'n'** to finish ordering
* View your final bill
